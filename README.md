# Excel Sales Analysis Project

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts

## Project Overview
This project analyzes sales data to understand sales performance, trends, and patterns.
The dashboard was created using pivot tables and charts for better business insights.

## Key Insights
- Monthly sales trend analysis
- Top-performing products
- Region-wise sales comparison
- Sales performance summary

## Files Included
- Excel sales analysis file (.xlsx)
